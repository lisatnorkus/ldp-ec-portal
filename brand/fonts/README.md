# Fonts — LDPEC

Inter, loaded from Google Fonts via the `@import` in `colors_and_type.css`. Weights used: 400 / 500 / 600 / 700 / 900.

The system-prompt style guide nudges away from Inter as overused — but the LDPEC brand doc specifies Inter explicitly, and matching the existing portal codebase matters more than aesthetic novelty here. Sticking with Inter.

## Stack

```
font-family: "Inter", -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;
```

## Mono

System mono stack only — `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Courier New", monospace`. No web mono font loaded.
